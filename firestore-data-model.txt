# Home

Featured artworks

Query:

```dart
.where('published', isEqualTo: true)
.where('featured', isEqualTo: true)
.orderBy('featuredOrder')
```

---

Collections

```dart
.where('published', isEqualTo: true)
.orderBy('displayOrder')
```

---

Catalog

```dart
.where('published', isEqualTo: true)
.orderBy('createdAt', descending: true)
```

---

Artworks by Collection

```dart
.where('published', isEqualTo: true)
.where('collectionIds',
       arrayContains: collectionId)
```

---

Artworks by Artist

```dart
.where('published', isEqualTo: true)
.where('artistId',
       isEqualTo: artistId)
```
